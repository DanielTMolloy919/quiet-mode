# Quiet Mode

A browser extension that helps you focus by hiding distracting elements on YouTube.

## Installation

```bash
# Install dependencies
pnpm install

# Build
pnpm build
# Load unpacked extension from .output/chrome-mv3 (Chrome)

pnpm build:firefox
# or .output/firefox-mv2 (Firefox)
```

## Development

```bash
pnpm dev
pnpm dev:firefox
```
